# MKVIM

Personal configuration for Nvim
---

## Instalation

---

1. Install Nvim
As a first step, we need to have NeoVim.

### Windows (winget)

```bash
winget install Neovim.Neovim
```

### Debian-based Linux

```bash
sudo apt install neovim
```

### Arch Linux

```bash
sudo pacman -S neovim
```

### Verification

Run this to verify that NeoVim is successfully installed:

```bash
nvim --version
```
